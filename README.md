# GRC & Information Security Portfolio

Welcome to my governance, risk, and compliance (GRC) portfolio. This repository contains sanitized, real-world security documentation and framework mappings designed to align organizational processes with global standards like ISO/IEC 27001.

## 🌟 Key Achievements
* **19+ InfoSec Policies Developed:** Designed a comprehensive Information Security Management System (ISMS) policy pack for non-governmental organizations (NGOs).
* **Security Gap Analysis:** Conducted baseline security posture assessments for two organizations against ISO 27001 Annex A controls.

## 📂 Repository Structure
* `/Policies/` - Sanitized policy templates (Acceptable Use, Access Control, Incident Response).
* `/Templates/` - Gap analysis and Risk Assessment templates.
